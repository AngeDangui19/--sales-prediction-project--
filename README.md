# Sales Data Analysis Project

This project was done as a small personal project to practice
data analysis and machine learning with Python.

The goal was to analyze sales data stored in a SQLite database
and apply basic machine learning methods.

## What I did
- Created a SQLite database with synthetic sales data
- Performed exploratory data analysis using pandas
- Applied K-Means clustering to segment products
- Built a simple regression model to predict monthly sales

## Tools
- Python
- SQLite
- pandas
- scikit-learn
